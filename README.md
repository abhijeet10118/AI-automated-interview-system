# 🎯 PrepAI — AI-Powered Interview Preparation Platform

> Practice smarter. Get hired faster.

**PrepAI** is a full-stack AI interview preparation platform that generates personalized interview questions, conducts mock interview sessions with voice support, and provides instant AI feedback on your answers.

🔗 **Live Demo:** [https://ai-automated-interview-system.vercel.app/home](https://your-vercel-url.vercel.app)

---

## ✨ Features

| Feature                   | Description                                                               |
| ------------------------- | ------------------------------------------------------------------------- |
| 🤖 AI Question Generation | Role-specific questions generated by Groq LLaMA 3.3                       |
| 📄 Resume Analysis        | Upload your PDF — AI extracts skills and generates personalized questions |
| 🎤 Voice Interview Mode   | Speak your answers using browser-native speech recognition                |
| 📊 AI Feedback Scoring    | Instant scoring on clarity, technical depth & communication               |
| 📈 Analytics Dashboard    | Score trends, skill radar chart and weak topic detection                  |

| 🌙 Beautiful UI           | Glassmorphism design with animated star background                        |

## 🛠 Tech Stack

### Frontend

- React + Vite
- Tailwind CSS
- Recharts (analytics)
- Axios
- React Router DOM
- EmailJS (OTP delivery)

### Backend

- Django REST Framework
- SimpleJWT (authentication)
- Groq API (LLaMA 3.3-70b)

### Deployment

- Frontend → Vercel
- Backend → Render
- Database → Neon (PostgreSQL)

## 🚀 Getting Started

### Prerequisites

- Python 3.11+
- Node.js 18+
- PostgreSQL database (Neon recommended)

### Backend Setup

```bash
cd backend
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt
```

Create a `.env` file in the backend folder:

```env
SECRET_KEY=your-django-secret-key
DEBUG=True
DATABASE_URL=your-neon-postgresql-url
GROQ_API_KEY=your-groq-api-key
```

Run migrations and start server:

```bash
python manage.py migrate
python manage.py runserver
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Create a `.env` file in the frontend folder:

```env
VITE_API_URL=http://127.0.0.1:8000/api
```

## 🔑 API Endpoints

| Method | Endpoint                         | Description                     |
| ------ | -------------------------------- | ------------------------------- |
| POST   | `/api/users/register/`           | Register new user               |
| POST   | `/api/users/login/`              | Login & get JWT tokens          |
| POST   | `/api/users/verify-otp/`         | Verify email OTP                |
| POST   | `/api/questions/generate/`       | Generate AI questions           |
| POST   | `/api/interviews/start/`         | Start interview session         |
| POST   | `/api/interviews/{id}/submit/`   | Submit answer + get AI feedback |
| POST   | `/api/interviews/{id}/complete/` | Complete interview              |
| GET    | `/api/interviews/analytics/`     | Get score analytics             |

## 🧠 How AI Feedback Works

When you submit an answer, the backend sends your question + answer to **Groq LLaMA 3.3** with a structured prompt. The AI returns scores (1-10) for:

- **Clarity** — how clearly you explained your answer
- **Technical Depth** — accuracy and depth of technical knowledge
- **Communication** — how well you'd communicate this in a real interview

Plus specific improvement suggestions written in plain English.
